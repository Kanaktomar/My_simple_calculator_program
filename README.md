# My_simple_calculator_program

“A collection of my programming assignments and activities for Unit 5 – Emerging Trends in Programming.”

## How to Run
1. Open the program in your IDE (VS Code or Code::Blocks).
2. Run the file using the terminal or build/run button.
3. Enter numbers and choose an operation.

## Features
- Addition
- Subtraction
- Multiplication
- Division

## Program Flow
1. The user chooses an operation.
2. The user enters two numbers.
3. The calculator performs the selected operation.
4. The result is displayed on the screen.

## Example Output
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): *
Result: 50
